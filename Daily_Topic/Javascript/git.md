### 1，从暂存区撤销
```js
三种情况：
- 文件已经修改但是还没有添加到暂存区
  这种情况使用 `git checkout -- [文件名]`可以完成文件的撤销
- 文件修改了,已经添加到暂存区了,并且后面再没有修改过
  这种情况`git reset 文件名` 撤销 `git add` 操作,然后使用`git checkout  --文件名` 撤销在工作区的修改
- 文件修改了,已经添加到暂存区了,并且后面也已经修改过
  这种情况使用`git checkout  --文件名` 撤销在工作区的修改, 接着使用`git reset 文件名`撤销`git add` 操作,
  然后使用`git checkout  --文件名` 撤销在工作区的修改

```
